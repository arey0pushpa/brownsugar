The History of Linux began in 1991 with the commencement of a personal project by a Finnish student, Linus Torvalds, to create a new free operating system kernel. When Linus Torvalds first created Linux in 1991, he needed some basic tools to make it useful. For this, he turned to the GNU project that, under the leadership of Richard Stallman, had produced free versions of essential computing utilities such as a command shell (bash) and C compiler (gcc). Windows 8 vs Ubuntu



Unity Vs modern ui

Both Microsoft and Canonical have received considerable flak for the default user interfaces. In Microsoft’s case, of course, it’s the Modern UI, formerly known as Metro; in Canonical’s case, it’s Unity. Both are designed with touchscreens in mind, and borrow heavily from the mobile world.

Cloud integration

Starting with the launch of Ubuntu One in 2009, the cloud has played a key role in Ubuntu Linux for some time, enabling users to store files online (5 GB Free, There after $3/20G Data ) and sync them among computers and mobile devices, as well as to stream audio and music from the cloud to mobile devices.

Hardware support

Ubuntu’s requirements, are much more modest: Canonical recommends 512MB of RAM, plus 5GB on the hard drive. You’ll also find versions such as Lubuntu and Xubuntu for lower-spec machines. In short, if hardware is a constraining factor for you, Ubuntu is most likely the better choice.

Security

Although Windows RT apps run within a sandboxed environment for greater security, Windows 8 Pro desktop legacy apps have no equivalent. Instead, third-party software developers are left to their own devices to add security measures to their apps.Ubuntu Linux provide their own firewalls.

Admin tools

For administrative controls, Windows provides Active Directory, using dedicated Active Directory servers. Canonical supports Active Directory as well, and Ubuntu Linux clients can join to an Active Directory Domain using third-party software such as Likewise Open or Centrify.

Price

Ubuntu Linux is free, while Windows 8 Pro will reportedly cost $199 after the current introductory upgrade offer of $39 to $69 expires. Ubuntu Linux 12.10 could provide a compelling alternative. If nothing else, it’s almost certainly worth your while to try it online or take it for a free test drive.
